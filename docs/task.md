# 任务书要点摘要（task.md）

> 实现速查摘要，**权威原文**为仓库根 `一骑红尘：荔枝争运战 参赛选手任务书.md`。冲突以原文为准。

## 目标与胜负（§1、§7、§9）
- 600 结算帧限时贡运对抗；从 S01 岭南出发运荔枝到 S15 兴庆宫。最终总分高者胜，相同判平。
- 开局：好果 100、坏果 0、鲜度 100、小分队人手 8、护卫行动点 4。
- 交付条件：位于 S15 + 已宫门验核 + 好果>0 + 鲜度>0 + 非移动/窗口/休整/强制通行/处理/验核中。
- 得分项（向下取整）：
  - 送达基础分 `min(240, 120 + floor(任务基础分累计 × 4/3))`（需交付）
  - 好果数量分 `floor(剩余好果/100 × 180)`
  - 鲜度品质分 `floor(剩余鲜度/100 × 180)`
  - 用时分 `floor((600−交付时间)/600 × 70) × min(任务累计,90)/90`
  - 皇榜任务分 `min(180, 任务累计 + 里程碑)`；里程碑 <60→0,60-89→15,90-109→35,≥110→50
  - 破关悬赏分 `min(原始,80)+20`（有悬赏时）
  - 惩罚：普通非法前 5 免、第 6 起每次 −1（≤20）；交付后违规每次 −5（≤30）
- **未交付**：送达/好果/鲜度/用时=0；任务分 `min(累计,80)`；悬赏 `min(原始,25)`。
- 失联退赛：未交付队连续 10 帧缺动作=警告，连续 60 帧缺动作=退赛。

## 地图与移动（§2）
- 固定：S01 起点、S14 宫门、S15 终点；节点 S01-S15；路线类型 ROAD/WATER/MOUNTAIN/BRANCH。**其余（坐标/路线距离/资源/任务/障碍分布）为可变项，须从 start 读取**。
- 到站所需移动量 = `ceil(路线距离 × 路线耗时系数)`；耗时系数：ROAD 1380 / WATER 1250 / MOUNTAIN 1780 / BRANCH 1550。
- 每帧移动量 = `floor(基础每帧移动量 × 1000 / 天气通行倍率)`；基础：无 1000 / 快马 1200 / 短程马 1150 / 疾行令 1300。
- 天气通行倍率：正常 1000；暴雨命中水路 1350；山雾命中山路 1100。
- 每帧鲜度损耗 = 基础 × 天气系数 × 急策系数。基础：停靠/等待/处理 0.05；ROAD 0.055/WATER 0.045/MOUNTAIN 0.07/BRANCH 0.065。天气系数：酷暑 ×1.5、暴雨区内 ×1.3、山雾 ×1；急策：疾行令 ×1.25、护果令 ×0.2。
- 单向边只按方向；路线边上禁原路倒退，可 WAIT 悬停或改道（进度归零）。

## 资源（§3.3）
ICE_BOX 鲜度+10(上限100，鲜度0时拒绝) / FAST_HORSE 20帧 / SHORT_HORSE 14帧（马类不叠加、与疾行令互斥）/ BOAT_RIGHT 仅标记不可主动用 / PASS_TOKEN,OFFICIAL_PERMIT 付验牒 / INTEL 探路标记(距离≤15，仅停靠时用)。护卫行动点付兵争牌。

## 皇榜任务（§5）
T01/T02/T04/T06/T08/T11=30 分，T12/T13/T14=15 分；处理帧数见原文。T04 用 CLAIM_TASK 完成才得分并清障；从相邻节点亦可。提交 CLAIM_TASK 用任务实例 taskId。

## 窗口争夺（§5.4）
3 拍窗口（T+1/T+2/T+3 结算）。类型 RESOURCE/TASK/GATE/DOCK/PASS/OBSTACLE。
牌克制：YAN_DIE>QIANG_XING，QIANG_XING>XIAN_GONG，XIAN_GONG>YAN_DIE，XIAN_GONG>BING_ZHENG，BING_ZHENG>YAN_DIE&QIANG_XING，QIANG_XING…（详见原文 §5.4.4 矩阵）。3 拍胜点高者得权益；平局双方休整、对象冷却（≤2 次平局）。
牌成本：YAN_DIE 1 文书 / QIANG_XING 马类或疾行令(有增益免) / XIAN_GONG 鲜度≥80+1好果 / BING_ZHENG 1 护卫点 / ABSTAIN 免费。

## 拦截突破（§6）
- 障碍：CLEAR(占1好果,6帧,不算T04) / T04 / SQUAD_CLEAR / FORCED_PASS / 改道。
- 设卡 SET_GUARD：4 帧；防守值 `min(上限, 2+额外好果×2)`；每队最多 2 个有效设卡；随帧风化 −1。
- 攻坚 BREAK_GUARD：目标为相邻敌方有效设卡；好果 2/篓、坏果 3/篓攻坚值；破关令 +3。攻坚值≥防守值成功清零，否则失败休整 5 帧。
- 强制通行 FORCED_PASS：遇设卡触发 PASS 窗口；总帧数=路线耗时+时间税；时间税按防守值/障碍(固定8)计算。
- 破关悬赏：普通 10 / 关键关隘 18；触发条件见 §6.3.3。

## 宫宴冲刺与终局急策（§6.5）
- 冲刺（phase=RUSH）触发：390 帧后满足距离/到位条件，或 450 帧必触发。开放宫门验核；禁止新派小分队。
- 终局急策每队整局限 1 次：疾行令(2好果,15帧+30%移动,鲜度×1.25) / 护果令(30帧鲜度×0.2) / 破关令(绑定 BREAK_GUARD/VERIFY_GATE)。
